# Pull Request Templates

## Standard Feature PR

```markdown
## Summary
Brief description of changes

## Changes
- Change 1
- Change 2

## Testing
- [ ] Tests pass
- [ ] Manual testing completed

## Checklist
- [ ] Code follows style guidelines
- [ ] Self-review completed
```

## Bug Fix PR

```markdown
## Bug Description
What was broken?

## Root Cause
Why did it happen?

## Fix
How was it fixed?

## Testing
Steps to verify the fix
```

## Documentation PR

```markdown
## Changes
What docs were updated?

## Preview
Screenshots or examples

## Checklist
- [ ] Spelling/grammar checked
- [ ] Links verified
```
